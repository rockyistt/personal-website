## 🚀 快速开始

### 快速启动（推荐）

**Windows:**
```bash
setup.bat
```

**macOS/Linux:**
```bash
bash setup.sh
```

### 手动启动

```bash
# 1. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 2. 安装依赖
pip install -r requirements.txt

# 3. 初始化数据库
flask --app run init-db

# 4. 添加数据
flask --app run seed-db

# 5. 运行应用
python run.py
```

访问: **http://127.0.0.1:5000**
